Cascading Effect with CSS

When you set a property of a selector as 'p' with a certain value, the value applies to todas labels p. However, if you change the same property to a different value for a more specific instance of p, that change will prevail on the 'general rule'.

  • If you say: p {font-family: Garamond}, all 'p' will Garamond font.

  • BUT if you say li p {font-family: Verdana}, the 'p' out of 'li' will INSIDE the 'li' will Garamond font, and 'p' Verdana font.

  • Among more specific your selectors, CSS will give more importance to the style that you apply!

Greater specificity makes CSS to give priority to that particular style.
